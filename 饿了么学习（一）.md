# 饿了么学习（一）

## 项目构建

安装vue脚手架

`npm install -g vue-cli`

项目初始化

`vue init webpack <项目名>` 

其他略：webpack配置需单独记录学习

## stylus

本项目学习使用stylus<br>
安装 `stylus`  并根据提示安装依赖的加载器

## 后台数据模拟

    var goods = appData.goods
    var ratings = appData.ratings
    var apiRoutes = express.Router()
